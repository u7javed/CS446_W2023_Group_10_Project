# CS446 (Software Design and Architectures) W2023 Group_10 Project - GenS
This is the repository for the app Group 10 of CS 446/CS 646/ECE 452 Winter 2023 made.

The app is written in Kotlin, and it's a tool for generating a study plan according to the user's availability and preferences.
